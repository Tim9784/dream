# Морской бой (онлайн)

Веб-игра на двоих: комнаты с 6-значным кодом, расстановка кораблей, классические правила.

## Сервер

- URL: `http://135.106.161.125:8090`
- Код: `/opt/minecraft-panel/app.py`
- Redis DB 1 для комнат
- Minecraft сервис остановлен и отключён из autostart (файлы не удалены)

## Правила

- Поле 10×10
- Флот: 4, 3, 3, 2, 2, 2, 1, 1, 1, 1
- Корабли не касаются (включая углы)
- После попадания ход сохраняется
