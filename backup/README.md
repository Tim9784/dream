# Бэкап сайта Omove.ru

Полная копия игрового сайта (код + статика + шаблоны + конфиги деплоя).

## Содержимое

- `omove.ru/` — бэкап сайта
- Источники: ветка `cursor/omove-deploy-fb1b`, продакшен Masterhost (`omove.ru`), тестовый Selectel

## Что не включено

- `venv/`
- `__pycache__/`
- runtime-данные комнат (`data/store/`)
- временные файлы

## Восстановление

Скопировать содержимое `omove.ru/` в корень веб-сайта, поставить зависимости из `requirements.txt`, настроить `config.json` и перезапустить приложение (Passenger/gunicorn).
